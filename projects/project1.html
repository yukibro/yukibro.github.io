📌 프로젝트 개요

딥러닝 기반으로 Sparse Sensor만을 활용한 Full-body Pose Generation을 수행한 VR Chat 프로젝트입니다.

🔧 사용 기술

  • Unity + SteamVR
  • Python (PyTorch)
  • SMPL 모델

📂 주요 기능

  • 3개 센서 기반 포즈 예측 (HMD, 좌/우 컨트롤러)
  • 사용자 체형 보정 및 아바타 대응
