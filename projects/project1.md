# VR Chat Project (with Deep Learning)

## 📌 개요
딥러닝 기반 Sparse Sensor(HMD + 컨트롤러 2개)로 전신 포즈를 예측하고, 사용자 체형에 맞게 리타게팅한 VR 프로젝트.

## 🔧 기술 스택
- Python (PyTorch)
- Unity + SteamVR
- SMPL 모델
- ZeroMQ / TCP 소켓 통신
